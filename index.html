File Size

Optimizing SVG for the most savage of frontiers — the Wild Wild Web

SVG's are inherently much smaller than their rasterized cousins. This is due to the fact that, rather than declaring what color every pixel in an image is, the SVG file declares points and curves. Then, it strokes paths or fills shapes with color. This is more intensive on the CPU, but results in a much smaller filesize.

compare basic SVG file with a JPG file

If you are proficient with Inkscape, Illustrator, or Sketch, you can further downsize an SVG's filesize by minimizing the complexity that it has. Removing the number of waypoints reduces the complexity immensely.

compare basic SVG file with reduced file

You can further reduce an SVG file by running it through an optimization program, like SVGO. Be careful, though — if you reduce the complexity too much, you may make it more difficult to customize later on!
